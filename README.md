# ASSIST Physiological Monitoring Dashboard

A real-time physiological signal monitoring dashboard built in Python.

## Features

- Bluetooth Low Energy device scanning
- Nordic UART communication
- Real-time ECG visualization
- Electrodermal Activity (EDA) signal processing
- Accelerometer monitoring (X, Y, Z)
- CSV data logging
- Live signal console

## Technologies

Python  
PyQt5  
PyQtGraph  
BLEAK  
SciPy
